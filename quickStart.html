1 Run the server

cd /path/to/build-dir
clang-tags-server

2 Build the compilation database

make clean
clang-tags trace make

rm -f main.o
g++ -c -o main.o ../src/main.cxx
g++ -o main main.o

3 Load the compilation database

clang-tags load

Server response:

4 Index the source files

clang-tags index

Server response:

-- Indexing project
../src/main.cxx:
  parsing...    0.747997s.
  indexing...   0.022584
0.877169s.

5 Find symbol definitions

Find the definition location of the identifier located in main.cxx at offset 849: 

clang-tags find-def -r ../src/main.cxx 849

Server response:
-- display -- MemberRefExpr display
   /home/francois/projets/git/clang-tags/build/tests/src/main.cxx:15-17:8-3: CXXMethod display
   USR: c:@S@MyClass>#d@F@display#

6 Find symbol references

Find all uses of the Cursor::location() method (identified by its USR, as given in the second result of clang-tags find-def above): 

clang-tags grep 'c:@S@MyClass>#I@F@display#'
